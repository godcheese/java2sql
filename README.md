## Java2Sql
- java2sql-cli
- 用Java代码生成SQL语句
- 目前只支持MySQL

- 适配器 MySqlAdapter
- 目前组件 Migration（表管理）、Seeder（数据管理）

## Quick Start

#### Action
- migration:UsersMigration 生成 Users 表迁移 class
- migrate:UsersMigration 迁移 Users 表
- seed:UsersSeed 生成 Users 表 填充 class
- seeder:UsersSeed 填充 Users 表数据
